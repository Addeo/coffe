## 💡 БЫСТРЫЕ КОМАНДЫ

### Пересобрать всё:

```bash
# Angular + Android
cd frontend
npm run build:android
cd android
./gradlew assembleDebug
```

### Только web:

```bash
cd frontend
npm run build
```

### Только Android (если web уже собран):

```bash
cd frontend/android
./gradlew assembleDebug
```
