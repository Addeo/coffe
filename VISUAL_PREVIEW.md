## 🎯 User Flow

### Инженер заходит на страницу:

1. **Видит статистику заработка** (только свою)
   - За работу: 50,000 ₽
   - За авто: 5,000 ₽
   - Всего: 55,000 ₽
   - Заказов: 5

2. **Если есть непринятые заявки** → видит предупреждение
   - Нажимает "Посмотреть"
   - Видит только свои ASSIGNED заявки

3. **Может свайпнуть** для смены месяца
   - Влево → следующий месяц
   - Вправо → предыдущий месяц

4. **Может свернуть** статистику по заказам
   - Больше места для таблицы

### Админ заходит на страницу:

1. **Видит общую статистику** (всех инженеров)
   - За работу: 150,000 ₽
   - За авто: 15,000 ₽
   - Всего: 165,000 ₽
   - Заказов: 15
   - **Часов: 120** (дополнительная карточка)

2. **Видит количество** непринятых заявок
   - "У вас 3 непринятых заявок"

3. **Все те же возможности** что у инженера
   - Навигация по месяцам
   - Сворачивание секций
